Responsive Image Adjusting by Breakpoints

This demo shows image-set() CSS can work with existing media queries. Using breakpoints within the image-set helps decide which image should be shown at various widths. Breakpoints are useful for static images (images which cannot be dynamically resized by the server).

The primary content on the left side takes up 70% of the available width. The side content on the right side takes up 30% of the available width.

Adjusting the width of the browser will request various static images.
